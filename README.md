## Файл настроек
Для корректно работы необходим файл настроек. Создать его необходимо в корневой папке приложения. Имя файла должно быть `settings.json` Пример ниже.
```
{
    "store": {
        "user": "",
        "password": "",
        "url": ""
    },
    "service": {
        "port": 31337
    }
}
```
